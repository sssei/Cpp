## How To Use
### Build
```sh
$ mkdir build
$ cd build
$ cmake ..
$ make
```

Then executable objects are generated build/day1/, build/day2/, ...

Also you can build all source files in one directory by designating target like below.

```sh
$ make day1
```

### Execute
For example, to execute list1-1 in day1
```sh
./day1/list1-1
```
